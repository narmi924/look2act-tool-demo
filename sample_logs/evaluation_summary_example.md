| Metric | Value |
| --- | --- |
| Dataset users | 31 |
| Dataset devices | 16 |
| Valid samples | 7,395 |
| Fixed test mean angular error | 2.8206 deg |
| 31-user leave-one-user-out mean angular error | 2.4788 deg |
| 31-user leave-one-user-out mean screen-space error | 321.3820 px |

Notes:

- The public artifact does not include raw participant images.
- The public artifact does not include full dataset files.
- The public artifact does not include model weights, ONNX files, or `.pth` checkpoints.
- These numbers are aggregate demonstration evidence, not a state-of-the-art claim.
