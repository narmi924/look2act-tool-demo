# v0.1.0 Initial Submission

This release note describes the initial non-code demonstration artifact for the Internetware 2026 Tool Demonstration submission of Look2Act.

## Included

- Documentation package.
- Demonstration workflow.
- Security and privacy notes.
- Model and data availability notes.
- Release boundary.
- Limitation notes.
- Project summary.
- Anonymized schemas.
- Synthetic sample logs.
- Screenshot and screencast artifact entries.
- English aggregate evaluation figures.

## Not Included

- Source code.
- Full dataset.
- Raw eye-region or face images.
- Model weights.
- ONNX files.
- `.pth` files.
- Windows executable package.

## Notes

The Windows demonstration package is not included in the initial submission artifact.
